## query_api

Sends a prompt to the Groq API and retrieves AI-generated financial analysis.

### Function Signature

```python
def query_api(prompt: str)
```

### Parameters

<ParamField path="prompt" type="str" required>
  The analysis prompt to send to the AI model. Typically generated by `build_prompt()`.
</ParamField>

### Returns

<ResponseField name="response" type="str">
  The AI-generated analysis text from the model, or an error message if the request fails.
</ResponseField>

### API Configuration

The function uses configuration from `config.py` and `settings.json`:

**Headers:**
```python
headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}
```

**Endpoint:**
- URL loaded from `SETTINGS["url"]`
- Model specified in `SETTINGS["model"]`

### Request Structure

The API request payload follows the chat completion format:

```python
payload = {
    "model": SETTINGS["model"],
    "messages": [
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": prompt}
    ]
}
```

Additional settings from `SETTINGS["custom-json"]` are merged into the payload.

### Response Parsing

The function extracts the message content from the API response:

```python
data = response.json()
return data["choices"][0]["message"]["content"]
```

### Error Handling

The function handles various error scenarios:

- **Network errors**: Connection timeouts or network failures
- **HTTP errors**: 4xx/5xx status codes from the API
- **Authentication errors**: Invalid or missing API key
- **Request exceptions**: Any `requests` library errors

All errors return a formatted error message:
```python
return f"❌ API request failed: {e}"
```

### Example Usage

```python
from groq_api import query_api
from prompt_builder import build_prompt
from stock_data import get_stock_history

# Get stock data and build prompt
history = get_stock_history("AAPL")
prompt = build_prompt("AAPL", history)

# Query the API
analysis = query_api(prompt)

if not analysis.startswith("❌"):
    print("Analysis received successfully")
    print(analysis)
else:
    print("API request failed")
```

### Source Code

```python
import requests
from config import API_KEY, SETTINGS

url = SETTINGS["url"]
headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

def query_api(prompt: str):
    try:
        payload = {
            "model": SETTINGS["model"],
            "messages": [
                {"role": "system", "content": "You are a helpful assistant."},
                {"role": "user", "content": prompt}
            ]
        }

        response = requests.post(url, headers=headers, json=payload | SETTINGS["custom-json"])
        response.raise_for_status()
        data = response.json()
        return data["choices"][0]["message"]["content"]
    except requests.exceptions.RequestException as e:
        return f"❌ API request failed: {e}"
```
