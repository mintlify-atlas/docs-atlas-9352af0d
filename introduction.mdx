# Welcome to FinFram Screener

**FinFram Screener** is a modular CLI tool for automated stock analysis and AI-powered forecasting. It pulls historical market data, builds structured prompts, and queries AI models via OpenAI-compatible APIs to generate forward-looking insights, ideal for due diligence, portfolio screening, or macroeconomic context.

## How It Works

1. **Enter a stock ticker** - The tool prompts you for a ticker symbol (e.g., AAPL, TSLA)
2. **Fetch historical data** - Retrieves 6 months of price history via yfinance
3. **Generate analysis** - Builds a structured prompt with key metrics and statistics
4. **AI forecasting** - Queries your configured AI model for insights and predictions
5. **Interactive chat** - Continue the conversation with follow-up questions

<Note>
  FinFram Screener works with any OpenAI-compatible API endpoint, including Groq, Together.ai, and other providers.
</Note>
